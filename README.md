# loan_approval_system-using-ml
A Machine Learning project that predicts loan approval status based on applicant details using Python, Pandas, and classification algorithms.
This project implements a Machine Learning based Loan Approval Prediction system. 
It analyzes applicant details such as income, employment status, and credit history 
to predict whether a loan should be approved or rejected. 
The project is built using Python, Pandas, NumPy, and Scikit-learn.
